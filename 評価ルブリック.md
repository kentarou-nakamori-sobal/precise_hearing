# 志向確認ヒアリング 評価ルブリック

## 基本方針

- **5段階固定**（3段階は差がつかない、7段階は中間で迷う）
- **方向統一**：すべて「5＝項目名が示す方向に強い」。離職リスク5は「高リスク」、管理職適正5は「高適正」
- **各段階に発言例・行動エビデンスを紐付け**。「印象」で点をつけない
- **中森さん・松山さんが個別採点 → 後で突合**。2点以上ズレた項目だけ根拠を持ち寄って議論

## ③ 離職リスク（複合スコア）

冒頭で挙げた9項目を**それぞれ1〜5で採点 → 平均**を全体スコアに。

| サブ項目 | 5（高リスク） | 3（中間） | 1（低リスク） |
| --- | --- | --- | --- |
| 年齢 | 転職市場価値が高い帯（20代後半〜30代前半） | 30代後半〜40代前半（転職検討はするが慎重） | キャリア終盤、転職コスト高 |
| 金銭満足度 | 「同年代と比べて低い」発言 | 「特に不満はない」程度。積極的な満足表明なし | 「妥当」「むしろ良い」と明言 |
| ポジション満足度 | 「役割が見えない」「燻っている」 | 現状を受け入れているが意欲表明もない | 役割への手応え・やりがいを語る |
| スキルアップ満足度 | 「成長実感がない」「停滞」 | 「まあまあ」「困ってはいない」程度 | 学べている・伸びていると明言 |
| 仕事満足度 | 業務内容への興味低下・退屈発言 | 淡々とこなしている。可もなく不可もなく | 仕事の面白さを具体的に語る |
| 居心地満足度 | 人間関係のストレス言及 | 特に不満も愛着もない、ドライな関係 | 居心地の良さ・人間関係の良さを言及 |
| 残業 | 月45h超／恒常的な負荷不満 | 月20〜30h、許容範囲だが負担感も時々 | 月20h未満、コントロール感あり |
| よそよそしさ | 雑談ゼロ／敬語が硬い／視線が合わない | 業務会話は普通、雑談は少なめで踏み込まない | 自然な対話、雑談が成立、本音っぽい話も出る |
| 評価納得度 | 「正当に見られていない」発言 | 「まあそんなものかな」程度の受容 | 評価プロセスへの信頼を明言 |

### 採点上の注意

- **「3」と「情報不足」を区別する**：該当発言が無かった場合は「3」ではなく **N/A（要追加質問）** とする。これをやらないと聞けていない項目が全部3で埋まり、結果として「リスクなし」に見えてしまう。
- **平均だけでなく「5フラグ」も別管理**：サブ項目に5が1つでもあれば要警戒として別管理。平均で薄まる凶悪リスクを拾うため。

## ⑤ 育成可能性（2面評価）

育成可能性は **受容姿勢（マインドセット）× ポテンシャル（地頭・吸収力）** の掛け算。片方だけでは育たない（姿勢が良くても伸びない／地頭が良くても聞かない）。**両軸を別個に採点し、低い方を最終スコアとする（ボトルネック方式）**。

| 点 | 受容姿勢のアンカー | ポテンシャルのアンカー |
| --- | --- | --- |
| 5 | フィードバックを自発的に求める／学習エピソード豊富／自己客観視ができる | 新領域の理解が早い／抽象化・他領域への応用ができる／好奇心の幅と深さがある |
| 4 | 弱みを認識し改善行動をとっている | 飲み込みが早い／教えたことを少し違う文脈に応用できる |
| 3 | 言われれば直す。自発的学習は限定的 | 説明された範囲は理解。応用は限定的 |
| 2 | 自己認識と他者評価のズレが大きい | 理解に時間がかかる／同じ説明を繰り返し必要 |
| 1 | 防衛的、フィードバックを拒絶 | 新しい概念の習得に大きな困難 |

**最終スコア = min(受容姿勢, ポテンシャル)**

### ヒアリングでポテンシャルを見るチェックポイント

ポテンシャルは業務観察が本筋だが、1時間のヒアリングでも以下から相当の情報が取れる：

- **話の論理構造**：因果関係・条件分岐をきれいに語れるか
- **抽象化**：自分の業務を「ひと言で言うと」整理できるか
- **質問への反応**：未経験領域の質問にも、推測込みで筋の通った返しができるか
- **言語化能力**：曖昧な感覚を構造化して話せるか
- **好奇心の幅**：「やってみたいこと」が現職周辺だけか、越境的か

## 人材マップ：3軸構造

人材マップは3つの軸で記述する。

- **軸A 貢献タイプ**：組織への貢献の仕方（役割ポジショニング）
- **軸B 動機タイプ**：本人の動機（動かし方・引き止め方）
- **軸C 離職リスク**：定着可能性（リスクスコア）

同じ「中核人材」でも動機が違えば動かし方が異なる。また「中核人材かつ離職リスク高」が最重要対処ケースになるため、離職リスクは軸Aから独立させてフラグ管理する。

### 軸A：貢献タイプ

| タイプ | 定義 |
| --- | --- |
| **中核人材** | 業務遂行の中核を担い、組織にとって不可欠。抜けると業務が止まる |
| **管理候補** | マネジメント適性あり。将来の管理職候補 |
| **スペシャリスト** | 特定領域の深い専門性で貢献。代替困難な技能を持つ |
| **変革推進役** | 新しい取り組みを自発的に推進。変化の起点になれる |
| **一般戦力（標準層）** | 日常業務を安定的にこなす標準層。中核ではないが組織運営の土台。多くの場合この層が最大ボリューム |
| **判定保留** | 入社が浅い／観察機会不足で情報が足りない。次回ヒアリングで再判定 |

※軸Aは原則1つに分類するが、**中核人材＋スペシャリスト**のように重なる場合は主・副の2つまで可。

### 軸B：動機タイプ

| タイプ | 特徴 | ヒアリングでの見分け方 |
| --- | --- | --- |
| **成長志向** | スキル・キャリアアップが主動機。挑戦を歓迎 | 「やってみたいこと」が具体的・拡張的。学習エピソード豊富 |
| **承認・地位志向** | ポジション・評価が主動機 | 「将来像」で肩書き・責任範囲を語る。評価への関心が高い |
| **貢献・使命志向** | 仕事の意義・顧客貢献が主動機 | 「良いところ」で理念や顧客の話が出る。要望も組織目線 |
| **安定志向** | 現状維持・予測可能性が主動機 | 変化への警戒、「今のままでいい」発言。要望が"守り"中心 |
| **生活重視（ライフ優先）** | 給与・労働時間が主動機。仕事は手段 | 「やってみたいこと」が薄い／プライベート話が中心。残業・金銭の話に反応強い |

### PMI対応方針（動機タイプ別）

| 動機タイプ | 拡大・変革局面での扱い方 |
| --- | --- |
| 成長志向 | 機会提供を最優先。放置すると外に機会を探す。新規プロジェクト・拡張役割を早めに割り当てる |
| 承認・地位志向 | ポジション付与で動く。肩書きが軽くなると離職リスク急上昇。新組織での処遇を明確に提示 |
| 貢献・使命志向 | 変革の「目的・意義」を語ることで動く。理念や顧客価値の言語化が効く |
| 安定志向 | 抵抗勢力化しやすい。**変化の必然性＋安心材料（雇用・処遇）を先に提示**。巻き込みは後半 |
| 生活重視 | 変革推進役にはならないが**戦力の安定供給源**。給与・労働時間が守られれば離職もしない。"巻き込む"のではなく"守る"対象 |

### 軸C：離職リスク

- 採点ルブリック③のスコア（1〜5）をそのまま使う
- 加えて、サブ項目に**5が1つでもあれば「5フラグ」**として別管理
- 軸A×軸Cの組み合わせで対処優先度を判定：

| 軸A × 軸C | 対処優先度 |
| --- | --- |
| 中核人材 × リスク4以上 | **最優先**。即時フォロー（1on1強化・処遇見直し） |
| 管理候補 × リスク4以上 | **高**。次期管理職パイプライン断絶のリスク |
| スペシャリスト × リスク4以上 | **高**。代替困難な技能の喪失リスク |
| 変革推進役 × リスク4以上 | **高**。PMI推進力の喪失リスク |
| 一般戦力 × リスク4以上 | **中**。複数同時離脱だけ注意（運営土台の崩れ） |
| 判定保留 × リスク4以上 | **要追加ヒアリング** |

### 運用上の注意

- **動機タイプは併存しうる**（成長×承認、貢献×安定など）。主・副の2つまで付けるくらいが運用上ちょうどいい
- ヒアリング1回では確定しないので、**初回は仮置き、観察で更新**前提
- 「生活重視＝悪い」ではない。**会社に必要な層**。ラベルにネガティブな意味を持たせないこと（評価ではなく対応方針のため）
- 「一般戦力」も同様。**評価が低いという意味ではなく、組織運営の土台**としての位置付け
