# 志向確認ヒアリング

## 目的

- 今後社員を増やし、会社を拡大するための（可能かどうかも含めた）情報を取得すること
- 管理職候補となる方のピックアップ、退職リスクのフォロー、社員の志向の確認

### 離職リスクとして特に確認しておきたい項目

※今回のヒアリングですべてを把握できなくても可

| 観点 |
| --- |
| 年齢 |
| 金銭満足度 |
| ポジション満足度 |
| スキルアップ満足度 |
| 仕事満足度 |
| 居心地満足度 |
| 残業 |
| よそよそしさ |
| 評価納得度 |

## 形式

- リモートで1時間

## ヒアリングを行う上での前提

- 本ヒアリングは評価の対象には一切しない
- ここで何を言ってもその言動により何かが不利になることはない

---

## ヒアリング内容

### 1. 志向に関する確認

- 現在の業務は？満足していますか？（金銭的、ポジション的、仕事的、居心地的、スキルアップ的）
- 今後やってみたいことはありますか？
- 将来どのようになっていたいですか？（エンジニア、リーダー、マネージャー、フルスタックエンジニア、別に何でも）

### 2. 変化に対する耐性を測る

- 今の会社の良いところは何だと思いますか？
- 逆に改善してほしいと思っているところ、会社への要望はありますか？

### 3. 不安解消目的

- 他に何か気になっていることや不安なことはありますか？

## 最終的に定量化したい項目

各項目の5段階採点基準は [`評価ルブリック.md`](./評価ルブリック.md) を参照。

| 項目 | 情報源 |
| --- | --- |
| 業務能力（テクニカル） | ヒアリング以外で情報を得る |
| 業務能力（ヒューマン） | ヒアリング以外で情報を得る |
| 管理職・リーダー適正 | ヒアリング |
| 変化・拡大適応性 | ヒアリング |
| 離職リスク | ヒアリング |
| 周囲への影響力 | ヒアリングとヒアリング以外の両方 |
| 育成可能性 | ヒアリングとヒアリング以外の両方（受容姿勢はヒアリング、ポテンシャルは業務観察も併用） |

## 最終的に策定していくもの

### A. 組織診断

- 拡大できるか
- 変化できるか
- 管理可能か

### B. 人材マップ（3軸構造）

各社員を以下の3軸で記述する。詳細な分類定義と採点基準は [`評価ルブリック.md`](./評価ルブリック.md) を参照。

- **軸A 貢献タイプ**：中核人材／管理候補／スペシャリスト／変革推進役／一般戦力（標準層）／判定保留
- **軸B 動機タイプ**：成長志向／承認・地位志向／貢献・使命志向／安定志向／生活重視
- **軸C 離職リスク**：1〜5スコア＋5フラグ（軸Aとの掛け合わせで対処優先度を決定）

### C. PMI戦略

- 拡大のスピード感
- まず安定化するか
- 制度変更どこまでやるか
- 誰を先に巻き込むか
